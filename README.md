# pandas-workflows
Four Pandas workflows: Titanic dataset, E-commerce dataset created from scratch, Supermarket Sales, and basic Pandas practice.

# Pandas Workflows – Hands-on Practice

This repository contains **four complete Pandas workflows** designed for practical learning and real-world scenarios. These notebooks cover **data cleaning, feature engineering, aggregation, visualization**, and more. They reflect workflows often used in **junior AI/data engineering roles** and are great for building a strong portfolio.

---

## 📁 Workflows Overview

### 1️⃣ Titanic Dataset
- **Notebook:** `01_titanic/titanic_workflow.ipynb`
- **Dataset:** [Kaggle Titanic](https://www.kaggle.com/c/titanic)
- **Description:**  
  A classic dataset to practice data preprocessing and analysis.  
  - Handling missing values (Age, Embarked)  
  - Encoding categorical variables (Sex, Embarked)  
  - Feature engineering (Family size, Title extraction)  
  - Aggregation and groupby analysis  
  - Sorting, filtering, and visualization  

**Skills Covered:** Data cleaning, feature engineering, grouping, aggregation, plotting.

---

### 2️⃣ E-Commerce Dataset (Created from Scratch)
- **Notebook:** `02_ecommerce/ecommerce_workflow.ipynb`
- **Description:**  
  A fully **synthetic dataset** created to simulate an e-commerce store.  
  - Generating a DataFrame from scratch  
  - Calculating totals, taxes, and profits  
  - Handling missing or inconsistent values  
  - Filtering, sorting, aggregation, and pivot tables  
  - Visualization (bar charts for sales by product)  

**Skills Covered:** Data creation, feature engineering, groupby, aggregation, pivot tables, plotting.

---

### 3️⃣ Supermarket Sales Dataset
- **Notebook:** `03_supermarket_sales/supermarket_workflow.ipynb`
- **Dataset:** [Supermarket Sales Kaggle](https://www.kaggle.com/datasets/faresashraf/supermarket-sales)
- **Description:**  
  Real-world supermarket dataset covering multiple branches.  
  - Data inspection and understanding  
  - Handling missing values  
  - Filtering by branches, cities, and product lines  
  - Groupby analysis to summarize sales and profits  
  - Pivot tables and multi-dimensional analysis  
  - Sorting, aggregations, and visualization (bar, line charts)  

**Skills Covered:** Data cleaning, aggregation, pivot tables, filtering, visualization.

---

### 4️⃣ Basic Pandas Operations
- **Notebook:** `04_basic_pandas/basic_pandas.ipynb`
- **Description:**  
  A practice notebook for **day-to-day Pandas operations** using a simple dataset.  
  - Selecting, filtering, and sorting rows and columns  
  - Handling missing values and duplicates  
  - Groupby and aggregation operations  
  - Creating new columns and basic feature engineering  
  - Simple visualizations for analysis  

**Skills Covered:** Core Pandas operations for real-world workflows.
